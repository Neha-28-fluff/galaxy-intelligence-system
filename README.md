# Galaxy Image Machine Learning Project 🚀

## Overview

This project explores machine learning and deep learning approaches applied to galaxy image data from astronomical surveys. The major goals are:
- **Classify galaxy images by shape** (spiral, elliptical, irregular)
- **Estimate galaxy redshift (distance) from images/data**
- **Detect anomalies in galaxy images**
- **Map galaxy images to their closest Pokémon by color and visual similarity (fun visualization feature)**

The project is an end-to-end demonstration: from data exploration, preprocessing, and model training to creative visualization and deployment.
